|------|-------------------|---------------------|----------------|

| **RAM Copy** | **2 copies**: `buffer` + decoded cache | **1 copy**: OS page cache (shared) | Buffered uses 2Ã— RAM |

| **View Creation** | References `self.buffer` | References `self.buffer` | Both zero-copy |
| **First Access** | ~7 CPU ops | ~7 CPU ops + page fault | Mmap has OS overhead |
| **Subsequent Access** | ~7 CPU ops | ~7 CPU ops | Mmap faster (no syscall) |
